# VS Code Settings

### Extensions Location
```bash
~/.vscode/extensions
```

### Generate Install Commands for Extensions in Bulk
```bash
code --list-extensions | xargs -L 1 echo code --install-extension
```

### Settings Stored
[Settings Sync](https://code.visualstudio.com/docs/editor/settings-sync)
